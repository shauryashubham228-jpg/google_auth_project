# Gmail OAuth Integration Project

A secure Django-based application that demonstrates OAuth 2.0 authentication with Google to fetch and display a user's most recent Gmail messages. This project highlights proficiency in Django development, Google API integration, and secure credential handling.

## 🚀 Key Features
*   **Google OAuth 2.0:** Implemented with PKCE (Proof Key for Code Exchange) for secure authorization flows.
*   **Gmail API Integration:** Fetches and parses the 5 most recent emails from the authenticated user's inbox.
*   **Data Processing:** Extracts key metadata (Sender, Subject, and Snippet) for a clean, user-friendly interface.
*   **Django Architecture:** Leverages Django's MVT pattern, session management, and template engine.

## 🛠 Tech Stack
*   **Backend:** Django 6.0
*   **Authentication:** `google-auth-oauthlib`, `google-auth-httplib2`
*   **API Client:** `google-api-python-client`
*   **Database:** SQLite
